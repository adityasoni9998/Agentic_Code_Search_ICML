\section*{Impact Statement}
Our work studies how to train language models to perform repository-level code localization, a key capability underlying many software engineering agent tasks like automated issue resolution. Advances in code localization will help strengthen end-to-end software engineering systems that rely on accurate retrieval of relevant code. Progress in automated coding agents has significant societal implications as they can reshape the software engineering profession, potentially reducing the demand for certain types of routine programming and maintenance tasks and contribute towards workforce displacement. We advocate for the responsible development and deployment of coding agents, with appropriate safeguards and human oversight to mitigate real-world harm.
